# House Price Prediction Model Report

## Objective
Predict house prices using machine learning techniques.

## Data Preparation
- Removed Property_ID
- Converted categorical variables using one-hot encoding
- Split dataset into 80% training and 20% testing

## Models Implemented
1. Linear Regression (from scratch)
2. Linear Regression (scikit-learn)
3. Random Forest Regressor

## Evaluation Metrics
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RÂ² Score

## Results
Linear Regression performed reasonably well.
Random Forest improved prediction accuracy.

## Insights
- Area strongly impacts price.
- Location significantly influences value.
- Tree-based models capture nonlinear relationships better.
